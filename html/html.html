    表格

    表格
    标签 描述 备注
    table 定义表格标签
    属性名 属性值 描述
    align left、center、right 规定表格相对周围元素的对应方式
    border 1或"" 是否拥有边框，默认为"",表示没有
    cellpadding 像素值 规定单元边沿与其他内容之间的空白，默认为1
    cellspacing 像素值 单元格之间的空白，默认2
    width 像素值或百分比 表格宽度
    border-collapse：collapse； (边框合并)
    让相邻表格边框进行合并，得到细线边框效果
    caption 表格的标题 必须紧随table标签
    tr 定义表格行 必须在table里面
    th 表头 居中，加粗
    td 表格中的单元格 必须在tr里面
    td标签可以包含任何内容
    rowspan 跨行合并单元格 目标单元格
    跨行：最上侧单元格为目标单元格，写合并代码
    跨列：最左侧单元格为目标单元格，写合并代码
    合并单元格步骤：1.先确定跨行还是跨列，2.找到单元格，写上合并方式=合并单元格数量。比如：<td colspan="2"></td>。3.删除多余的单元格
    colspan 跨列合并单元格
    thead 表格的头部 thead里面必须有tr标签
    tbody 表格的主体 tbody里面必须有tr
    tfoot 表格的尾部

    常用属性 属性值 作用

    标签 描述 属性
    input 输入表单元素
    属性 属性值 描述
    type button 配合value定义按钮内容,定义可点击按钮（用于通过JavaScript启动脚本）
    checkbox 定义复选框(多选框) name必须相同，才可以实现复选
    file 定义输入字段和“浏览”按钮，供文件上传
    image 定义图像形式的提交按钮

    label

    label
    标签 描述 备注 用法 如
    label 元素定义标注 绑定一个表单元素，点击时会自动将光标或焦点选择到对应的表单元素上，以增强用户体验
    label标签里必须有for属性，input标签里必须有id属性

    使用方法①：

    1. 使用label标签把内容（如：文本）包裹起来

    2. 在表单标签上添加id属性

    3. 在label标签的for属性中设置对应的id属性值

    使用方法②：

    1. 直接使用label标签把内容（如：文本）和表单标签一起包裹起来

    2. 需要把label标签的for属性删除即

    		 <label for="sex2">男</label>
    <input type="radio" name="sex" id="sex2">

    或

    <label><input type="radio" name="sex">男</label>
